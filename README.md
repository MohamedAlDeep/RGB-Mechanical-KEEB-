# RGB Mechanical 75% KEEB

This is my custom made 75% rgb mechanical keyboard, it has per-key lights and hot swapable key switches.

> Why I made this project:

Because i have a laptop and i haven't had any actual separate keyboard experience, so i wanted to have one, and thought it would be nice for me to make my own for flexing on my fellows and solidify my PCB desiging knowledge.
